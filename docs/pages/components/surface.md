# Surface
Similar to Cards in material design, used to display content raised off of the background.

## Usage

<kro-surface>Hey this is a surface</kro-surface>

```html
<kro-surface>Hey this is a surface</kro-surface>
```

## Props
```ts
padded: {
    type: Boolean,
    default: true
},

raised: {
    type: Boolean,
    default: true,
}
```
