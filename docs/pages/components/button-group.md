# Button Groups
Button groups provide a way for you to group multiple related buttons into a unified style.

## Usage
<kro-button-group>
    <kro-button>Cut</kro-button>
    <kro-button>Copy</kro-button>
    <kro-button>Paste</kro-button>
</kro-button-group>
