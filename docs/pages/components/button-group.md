# Button Groups
Button groups allow you to group and display multiple related buttons together.

## Usage
<kro-button-group>
    <kro-button>Cut</kro-button>
    <kro-button>Copy</kro-button>
    <kro-button>Paste</kro-button>
</kro-button-group>
